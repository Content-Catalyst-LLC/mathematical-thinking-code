# Mathematical Thinking Code

Open computational companion repository for the Mathematical Thinking knowledge series.

This repository supports articles on pattern recognition, abstraction, proof, representation, recursion, discrete mathematics, graph reasoning, mathematical history, and mathematical thinking for computer science.

## Repository Structure

- `articles/mathematical-thinking-pattern-proof-architecture-reason/`
  - `python/` — sequence, recursion, proof-graph, and graph-reasoning examples
  - `r/` — sequence pattern analysis and generalization workflows
  - `julia/` — recurrence and high-performance mathematical exploration
  - `sql/` — theorem, proof-step, concept, and example metadata schemas
  - `rust/` — command-line proof-pattern utility scaffold
  - `go/` — graph and pathway analysis scaffold
  - `cpp/` — efficient discrete-structure examples
  - `fortran/` — recurrence and numerical-sequence examples
  - `c/` — low-level sequence and graph utilities
  - `docs/` — article notes and model documentation
  - `data/` — small synthetic datasets
  - `outputs/` — generated outputs

## Purpose

The goal is not to replace proof with computation. The goal is to support proof-aware exploration: generating examples, seeing patterns, visualizing structure, testing conjectures, documenting assumptions, and distinguishing empirical evidence from mathematical justification.
