# Ethics Notes

Recursive systems can amplify structure, error, bias, and feedback.

## Responsible questions

- What is the base case?
- What rule is repeated?
- What measure decreases or changes?
- What stops the process?
- What is preserved?
- What is amplified?
- What happens if outputs influence future inputs?
- How is the recursive system externally validated?

Recursion should clarify structure, not hide compounding error behind repeated procedure.
