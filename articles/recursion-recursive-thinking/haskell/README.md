# Haskell Companion Scaffold

This Haskell folder supports **Recursion and Recursive Thinking**.

Haskell is especially useful for this article because recursive data types, pattern matching, structural recursion, and pure functions make recursive structure explicit.

## What this folder demonstrates

- recursive factorial and Fibonacci definitions;
- memoized Fibonacci with a map;
- recursive tree data types;
- expression trees and recursive evaluation;
- recursive-definition metadata;
- structural recursion over trees;
- CSV output for recurrence and structure audits.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_recursive_definition_audit.csv
../outputs/tables/haskell_recurrence_audit.csv
../outputs/tables/haskell_tree_expression_audit.csv
```
