# Data Notes

The datasets are synthetic recursive-structure examples and audit metadata.

- `raw/recursive_definitions.csv`: base cases, recursive rules, termination measures, and interpretations
- `raw/recurrence_cases.csv`: recurrence relations, closed-form notes, and proof status
- `raw/tree_nodes.csv`: proof-tree and expression-tree examples
- `raw/grammar_rules.csv`: recursive grammar rules
- `raw/algorithm_cases.csv`: recursive algorithm base cases, reductions, combination rules, and correctness notes
- `raw/model_update_cases.csv`: recursive state update examples and risks
- `raw/recursion_warnings.csv`: warnings and mitigations for recursive structures
- `processed/`: generated data products
