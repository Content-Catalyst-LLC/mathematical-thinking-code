# Validation and Verification Notes

Verification asks whether the model has been implemented correctly. Validation asks whether the model is adequate for its intended representation and use.

## Distinction

```text
verification: solving the model right
validation: using the right model
```

## Review questions

- Is the code correct?
- Are the numerical methods stable?
- Are equations solved accurately?
- Does the model match relevant observations?
- Does it represent plausible mechanisms?
- Is it adequate for the decision or purpose?
