# Scientific Modeling Notes

A scientific model is a structured representation of selected aspects of a target system.

## Review questions

- What target system is represented?
- What are the boundaries?
- What variables and parameters are chosen?
- What assumptions make the model possible?
- What data connect the model to observation?
- What is the intended use?
- What should not be inferred from the model?

A model is useful because it simplifies, but dangerous when simplifications are forgotten.
