# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, model indexes, assumption summaries, variable and parameter metadata, calibration summaries, validation summaries, uncertainty summaries, sensitivity summaries, model risk indexes, and responsible modeling checklists
- `outputs/figures/`: Graphviz DOT files and optional generated scientific modeling workflow diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
