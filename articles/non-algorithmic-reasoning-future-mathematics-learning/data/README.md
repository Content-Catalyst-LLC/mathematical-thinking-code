# Data Notes

The datasets are synthetic mathematics-learning metadata files.

- `raw/reasoning_moves.csv`: non-algorithmic reasoning stages and guiding questions
- `raw/learning_tasks.csv`: tasks with algorithmic and non-algorithmic components
- `raw/assessment_dimensions.csv`: rubric dimensions and scoring notes
- `raw/student_solution_audits.csv`: synthetic student reasoning scores
- `raw/ai_outputs.csv`: AI-generated-output audit examples
- `raw/misconceptions.csv`: misconception metadata and interventions
- `processed/`: generated data products
