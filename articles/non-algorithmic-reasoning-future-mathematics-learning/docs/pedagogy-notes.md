# Pedagogy Notes

This article treats non-algorithmic reasoning as a central component of mathematical learning.

## Key learning dimensions

- problem framing;
- representation choice;
- strategy explanation;
- assumption checking;
- verification;
- proof judgment;
- interpretation;
- reflection.

## Instructional principle

Procedural fluency matters, but it should be joined to conceptual understanding, strategic flexibility, proof literacy, and metacognitive self-monitoring.
