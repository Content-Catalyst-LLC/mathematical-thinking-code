# Assessment Notes

Assessment should include answer correctness, but not stop there.

## Reasoning dimensions

- framing;
- representation;
- strategy;
- assumptions;
- justification;
- reflection.

## Useful assessment prompts

- Explain why this method applies.
- State the assumptions and domain.
- Choose a representation and justify it.
- Check the answer in the original problem.
- Find or rule out a counterexample.
- Compare two solution strategies.
- Reflect on what transfers to a new problem.
