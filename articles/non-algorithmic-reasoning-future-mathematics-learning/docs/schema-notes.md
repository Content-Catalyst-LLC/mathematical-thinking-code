# Schema Notes

The SQL schema models mathematics-learning reasoning metadata.

## Core tables

- `reasoning_move`: non-algorithmic reasoning stages and guiding questions.
- `learning_task`: tasks with algorithmic and non-algorithmic components.
- `assessment_dimension`: rubric dimensions and evidence of learning.
- `misconception`: diagnostic signals and interventions.
- `ai_output_audit`: generated outputs requiring verification.
- `solution_audit`: scored reasoning evidence from synthetic student work.

## Professional use cases

- mathematics education research;
- task design;
- AI-assisted learning audits;
- proof-literacy assessment;
- representation-choice analysis;
- misconception diagnosis;
- reasoning rubric development.
