# Haskell Companion Scaffold

This Haskell folder supports **What Is Mathematical Thinking?**.

Haskell is included here because it is unusually well suited to Mathematical Thinking repositories that involve logic, proof, abstraction, recursion, type structure, algebraic data types, formal inference, and proof/computation relationships.

## What this folder demonstrates

- algebraic data types for propositions;
- truth tables for implication, converse, and contrapositive;
- proof-step and dependency records;
- finite sequence-pattern audits;
- simple graph-invariant calculations;
- pure functional transformations that make mathematical structure explicit.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_truth_table.csv
../outputs/tables/haskell_proof_dependencies.csv
../outputs/tables/haskell_sequence_audit.csv
../outputs/tables/haskell_graph_invariants.csv
```

## Professional note

These examples are designed as lightweight scaffolds for mathematical reasoning. They support formalization planning, proof literacy, functional modeling, and reproducible exploration. They do not replace proof, proof-assistant checking, peer review, or expert mathematical interpretation.
