# Data Notes

The datasets in this folder are synthetic mathematical metadata files used for reproducible demonstrations.

- `raw/sequence_catalog.csv`: example sequence definitions and interpretive notes
- `raw/concepts.csv`: concept-level metadata for mathematical-thinking structures
- `raw/theorems.csv`: theorem-level metadata
- `raw/proof_steps.csv`: proof-step records tied to the theorem table
- `raw/proof_dependencies.csv`: directed dependency edges among concepts and theorems
- `processed/`: generated tables created by the scripts
