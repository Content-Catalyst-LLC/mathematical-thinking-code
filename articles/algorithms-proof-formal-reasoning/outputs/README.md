# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, specification audits, invariant checks, postcondition checks, binary-search traces, complexity tables, proof-obligation indexes, termination indexes, graph-algorithm assumption indexes, evidence tables, and warning indexes
- `outputs/figures/`: Graphviz DOT files and optional generated diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
