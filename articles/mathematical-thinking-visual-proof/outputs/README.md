# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, visual proof indexes, diagram relation summaries, proof-without-words summaries, role/status summaries, risk indexes, accessibility reviews, workflow checklists, and domain review summaries
- `outputs/figures/`: Graphviz DOT files and optional generated visual-proof workflow diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
