# Haskell Companion Scaffold

This Haskell folder supports **Mathematical Thinking and Visual Proof**.

Haskell is useful here because visual roles, domains, proof statuses, and visual risks can be modeled explicitly with algebraic data types.

## What this folder demonstrates

- visual arguments as typed roles;
- visual domains as typed categories;
- proof status as explicit state;
- visual risks as typed records;
- human review as an explicit responsibility;
- see-abstract-prove-interpret as a reusable workflow.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_visual_proof_records.csv
../outputs/tables/haskell_visual_risks.csv
../outputs/tables/haskell_see_abstract_prove_interpret.csv
```

These examples support visual-proof literacy, diagrammatic reasoning audits, accessibility review, and responsible mathematical visualization. They do not replace mathematical proof, formal verification, expert review, accessibility review, classroom judgment, or ethical visualization practice.
