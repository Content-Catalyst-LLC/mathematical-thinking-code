# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, historical timelines, thinking-mode summaries, tradition/mode matrices, representation indexes, structural-abstraction indexes, computational milestone indexes, historiographic warnings, and responsible-history checklists
- `outputs/figures/`: Graphviz DOT files and optional generated historical diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
