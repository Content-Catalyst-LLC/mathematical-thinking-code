# Haskell Companion Scaffold

This Haskell folder supports **The History of Mathematical Thinking from Antiquity to Modernity**.

Haskell is useful here because historical mathematical modes, media, milestones, structures, and warnings can be modeled as explicit algebraic data types. That makes the article’s intellectual architecture inspectable: mathematical thinking becomes a typed model of periods, traditions, representations, proof styles, structures, and formalization.

## What this folder demonstrates

- thinking modes as typed alternatives;
- media of mathematical representation as typed alternatives;
- milestones as structured records;
- structural abstractions as typed records;
- historiographic warnings as explicit review objects;
- CSV output for reproducible audit artifacts.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_historical_milestones.csv
../outputs/tables/haskell_structural_abstractions.csv
../outputs/tables/haskell_historiographic_warnings.csv
```

## Professional note

These examples support mathematical-history pedagogy, structured historiography, proof/notation/computation mapping, and responsible canon review. They do not replace expert history of mathematics, primary-source scholarship, philology, mathematical proof, or formal verification.
