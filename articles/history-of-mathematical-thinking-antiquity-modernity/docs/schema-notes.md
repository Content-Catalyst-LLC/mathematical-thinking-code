# Schema Notes

The SQL schema models periods, traditions, thinking modes, milestones, representation forms, structural abstractions, computational milestones, and historiographic warnings.

## Core tables

- `historical_period`: broad period metadata.
- `mathematical_tradition`: tradition, context, medium, mode, and caution.
- `thinking_mode`: analytic categories for mathematical thought.
- `mathematical_milestone`: link between period, tradition, mode, contribution, representation, and significance.
- `representation_form`: media and symbol systems.
- `structural_abstraction`: modern mathematical structures.
- `computational_milestone`: algorithmic and computational developments.
- `historiographic_warning`: canon, power, presentism, and technological-triumphalism risks.

## Professional use cases

- mathematical-history teaching;
- structured historiography;
- knowledge-graph modeling;
- curriculum design;
- proof/notation/computation mapping;
- responsible canon review.
