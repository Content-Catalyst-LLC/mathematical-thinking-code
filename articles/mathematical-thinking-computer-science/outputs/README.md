# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, concept audits, algorithm specifications, complexity growth tables, proof obligations, automata audits, graph reachability tables, type audits, probability audits, linear-algebra representation audits, and representation warnings
- `outputs/figures/`: Graphviz DOT files and optional generated diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
