# Complexity Notes

Complexity analysis studies how resource use grows with input size.

## Common growth classes

- \(O(1)\): constant
- \(O(\log n)\): logarithmic
- \(O(n)\): linear
- \(O(n\log n)\): near-linear
- \(O(n^2)\): quadratic
- \(O(2^n)\): exponential

Complexity is a model of growth, not an exact runtime promise.
