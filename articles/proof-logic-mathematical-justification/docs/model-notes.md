# Model Notes

This article treats proof as accountable mathematical justification.

The companion repository models proof through six linked ideas:

1. **Claim** — the statement proposed for proof or refutation.
2. **Assumption** — the conditions under which the claim is evaluated.
3. **Inference** — a valid transition from premises to conclusion.
4. **Proof step** — an ordered unit of justification.
5. **Dependency** — a support relation among definitions, lemmas, rules, and theorems.
6. **Counterexample** — an object that refutes a universal claim.

The repository separates finite evidence from proof status. This is essential: examples can support conjectures, but proof carries the logical burden.
