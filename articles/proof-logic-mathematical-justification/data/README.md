# Data Notes

The datasets are synthetic mathematical proof metadata files.

- `raw/claims.csv`: definitions, lemmas, theorems, false conjectures, and inference rules
- `raw/assumptions.csv`: assumptions linked to claims
- `raw/proof_steps.csv`: ordered proof-step records
- `raw/proof_dependencies.csv`: directed dependency edges
- `raw/inference_rules.csv`: reusable inference-rule metadata
- `raw/counterexamples.csv`: counterexamples and theorem-refinement lessons
- `raw/finite_evidence.csv`: finite evidence records separated from proof status
- `processed/`: generated data products
