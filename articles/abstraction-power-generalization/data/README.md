# Data Notes

The datasets are synthetic mathematical metadata files used for reproducible demonstrations.

- `raw/fractions.csv`: rational representatives for equivalence-class analysis
- `raw/abstractions.csv`: abstraction records showing preserved and omitted structure
- `raw/generalizations.csv`: claims, domains, assumptions, and proof status
- `raw/graph_edges.csv`: graph examples for invariant analysis
- `raw/mappings.csv`: structure-preserving maps and warnings
- `raw/counterexamples.csv`: counterexample records linked to generalizations
- `processed/`: generated data products
