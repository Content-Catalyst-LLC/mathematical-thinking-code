# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, automation task indexes, tool literacy summaries, verification records, evidence-standard summaries, trust-boundary maps, risk indexes, judgment-skill indexes, proof-assistant layer audits, and specify-compute-verify-interpret checklists
- `outputs/figures/`: Graphviz DOT files and optional generated automation workflow diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
