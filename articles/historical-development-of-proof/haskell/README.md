# Haskell Companion Scaffold

This Haskell folder supports **The Historical Development of Proof**.

Haskell is useful here because proof styles, media, traditions, and historiographic warnings can be modeled explicitly with algebraic data types.

## What this folder demonstrates

- proof styles as typed alternatives;
- proof media as typed historical categories;
- traditions as structured records;
- historiographic warnings as explicit review objects;
- CSV output for reproducible audit artifacts.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_proof_traditions.csv
../outputs/tables/haskell_historiographic_warnings.csv
```

## Professional note

These examples support proof-history pedagogy, structured historiography, comparative proof-style modeling, and responsible canon review. They do not replace expert historical scholarship, primary-source analysis, philology, or mathematical proof.
