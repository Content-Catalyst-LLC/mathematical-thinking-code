# Schema Notes

The SQL schema models proof styles, proof traditions, proof milestones, proof media, proposition dependencies, and historiographic warnings.

## Core tables

- `proof_style`: proof-style classification metadata.
- `proof_tradition`: tradition, period, region/language context, dominant style, interpretation.
- `proof_milestone`: important texts, figures, periods, contributions, and significance.
- `proof_medium`: material and institutional media of proof.
- `proposition_dependency`: synthetic dependency graph for proof architecture.
- `historiographic_warning`: canon, presentism, translation, and technological-triumphalism risks.

## Professional use cases

- proof-history teaching;
- structured historiography;
- canon-risk auditing;
- proof-style comparison;
- cross-cultural mathematics mapping;
- formalization timeline modeling.
