# Data Notes

The datasets are synthetic algebraic-thinking metadata files.

- `raw/algebraic_objects.csv`: expressions, equations, functions, identities, sequences, and structures
- `raw/variable_roles.csv`: variable roles and interpretations
- `raw/transformation_rules.csv`: meaning-preserving and risky algebraic transformations
- `raw/equations.csv`: equation types and verification methods
- `raw/expression_examples.csv`: expression forms and visible structure
- `raw/function_representations.csv`: multiple representations of a linear function
- `raw/algebraic_misconceptions.csv`: misconception diagnostics and interventions
- `processed/`: generated data products
