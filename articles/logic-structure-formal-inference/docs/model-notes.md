# Model Notes

This article treats logic as the structure of valid inference.

The companion repository models formal inference through six linked elements:

1. **Proposition** — a statement with truth conditions in a logical setting.
2. **Predicate and quantifier** — open statements and variable-binding structures.
3. **Connective** — truth-functional structure such as conjunction, disjunction, implication, negation, and equivalence.
4. **Inference rule** — a valid transition from premises to conclusion.
5. **Derivation** — a structured chain or tree of inference-rule applications.
6. **Counterexample** — a domain object that refutes a universal claim.

The repository separates truth-functional validity from domain-specific mathematical proof. A truth table can verify a logical form, but a theorem about integers, functions, graphs, or models still requires appropriate mathematical justification.
