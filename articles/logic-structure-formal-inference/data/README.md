# Data Notes

The datasets are synthetic formal-logic metadata files.

- `raw/propositions.csv`: atomic, conditional, quantified, and false universal propositions
- `raw/inference_rules.csv`: reusable formal inference rules
- `raw/derivation_steps.csv`: ordered derivation-step examples
- `raw/derivation_dependencies.csv`: directed dependency edges for derivation graphs
- `raw/quantifier_patterns.csv`: quantifier forms with proof and refutation strategies
- `raw/counterexamples.csv`: refutation examples linked to proposition records
- `raw/proof_systems.csv`: proof-system comparison metadata
- `processed/`: generated data products
