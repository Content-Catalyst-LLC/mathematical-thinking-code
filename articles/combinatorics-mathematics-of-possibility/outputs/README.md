# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, counting audits, Pascal/binomial audits, recurrence tables, graph counts, partition counts, search-space growth, possibility warnings, probability audits, and method-validation indexes
- `outputs/figures/`: Graphviz DOT files and optional generated decision trees

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
