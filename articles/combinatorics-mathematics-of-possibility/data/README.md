# Data Notes

The datasets are synthetic combinatorial examples and possibility-space metadata.

- `raw/combinatorial_problems.csv`: structured counting problems with order/repetition/constraint assumptions
- `raw/counting_methods.csv`: method, formula, double-counting risk, and validation notes
- `raw/graph_count_cases.csv`: simple graph and labeled-tree counting cases
- `raw/probability_cases.csv`: sample-space probability examples and equal-likelihood notes
- `raw/recurrence_cases.csv`: recursive counting examples
- `raw/possibility_warnings.csv`: warnings and mitigations for interpreting possibility spaces
- `raw/generating_function_examples.csv`: generating-function examples and coefficient meanings
- `processed/`: generated data products
