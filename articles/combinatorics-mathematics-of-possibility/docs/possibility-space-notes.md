# Possibility-Space Notes

A possibility space is the set of valid configurations under stated rules.

## Required documentation

- object universe;
- allowed choices;
- constraints;
- distinctness rule;
- repetition rule;
- ordering rule;
- equivalence or symmetry rule;
- exclusion criteria;
- intended interpretation;
- probability assumptions, if any.

A possibility space is a model. It preserves selected structure and excludes other possibilities.
