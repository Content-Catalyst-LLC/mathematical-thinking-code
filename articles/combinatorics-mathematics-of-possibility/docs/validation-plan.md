# Validation Plan

## Counting assumptions

- Confirm whether order matters.
- Confirm whether repetition is allowed.
- Confirm whether objects are labeled or unlabeled.
- Confirm whether cases are disjoint.
- Confirm whether symmetries identify arrangements.

## Method selection

- Use combinations for unordered selection.
- Use permutations for ordered selection without repetition.
- Use multiplication principle for staged choices.
- Use inclusion-exclusion for overlap.
- Use recurrence for recursive construction.
- Use generating functions when coefficient encoding clarifies structure.

## Interpretation

- Separate count from probability.
- Separate possibility from likelihood.
- Record constraints and exclusions.
- Review scenario boundaries.
- Document when exhaustive search is infeasible.

## Responsible-use note

Synthetic examples in this repository are for methods demonstration. Applied use requires domain validation, stakeholder review, uncertainty documentation, and consequences analysis.
