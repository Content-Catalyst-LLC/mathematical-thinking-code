# Schema Notes

The SQL schema models combinatorial problems, counting methods, graph count cases, probability cases, generating-function examples, and possibility warnings.

## Core tables

- `combinatorial_problem`: problem definition and assumptions.
- `counting_method`: method, formula, double-counting risk, and validation note.
- `graph_count_case`: graph-counting examples and formulas.
- `probability_case`: sample-space and equal-likelihood checks.
- `generating_function_example`: coefficient-interpretation examples.
- `possibility_warning`: warnings and mitigations.

## Professional use cases

- combinatorics instruction;
- search-space audits;
- probability sample-space review;
- scenario enumeration;
- model assumption documentation;
- AI and optimization search-space governance.
