# Proof, Algorithm, and Model Notes

Proof, algorithm, model, simulation, and formal verification are related but distinct.

## Distinctions

- proof asks what follows from assumptions;
- algorithm asks what procedure transforms input to output;
- model asks what formal structure represents a target system;
- simulation asks what behavior appears under modeled dynamics;
- formal verification asks whether a formal derivation or system satisfies a specification.

Confusing these evidence standards can create false authority.
