# Counterexample Notes

Counterexample search is central to disciplined mathematical imagination.

A conjecture generated from pattern should be tested against:

- small cases;
- degenerate cases;
- boundary cases;
- infinite or limiting cases;
- nonstandard examples;
- structurally similar but non-equivalent objects;
- cases where a key hypothesis has been removed.

## Example

Claim: Graphs with the same degree sequence are isomorphic.

Problem: The degree sequence is an invariant, but it is not a complete invariant for graph isomorphism. Non-isomorphic graphs can share the same degree sequence.

Lesson: An invariant may distinguish some objects, but not fully classify the entire domain.
