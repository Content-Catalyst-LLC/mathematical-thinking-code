# Data Notes

The datasets are synthetic mathematical metadata files for reproducible demonstrations.

- `raw/pattern_sequences.csv`: sequence patterns and structural interpretations
- `raw/graph_objects.csv`: graph-level metadata
- `raw/graph_edges.csv`: edge lists for sample graphs
- `raw/invariants.csv`: invariant descriptions
- `raw/analogies.csv`: mathematical analogy records
- `raw/counterexamples.csv`: counterexample and failure-mode notes
- `processed/`: generated data products
