# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, discovery candidate indexes, evaluator summaries, verification records, candidate status summaries, human interpretation summaries, risk indexes, workflow audits, proof-status taxonomy, and responsible AI-discovery checklists
- `outputs/figures/`: Graphviz DOT files and optional generated AI-discovery workflow diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
