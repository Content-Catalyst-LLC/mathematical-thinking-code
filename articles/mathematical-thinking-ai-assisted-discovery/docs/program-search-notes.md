# Program Search Notes

Program search makes AI-assisted discovery more auditable because generated candidates can be executed and evaluated.

## Review questions

- What program was generated?
- What mathematical object or construction does it represent?
- What evaluator scored it?
- What constraints were checked?
- Does the code generalize beyond test cases?
- Can the behavior be explained mathematically?
- Can the result be converted into a theorem?

Program search can produce powerful candidates, but mathematical knowledge still requires proof and interpretation.
