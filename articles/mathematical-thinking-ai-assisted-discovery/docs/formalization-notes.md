# Formalization Notes

AI can assist formalization by suggesting syntax, theorem statements, tactics, or lemma names.

## Responsible workflow

```text
AI proposes -> proof assistant checks -> human interprets
```

## Review questions

- Does the formal statement match the intended informal theorem?
- Which hypotheses were added, omitted, or changed?
- Which libraries and axioms were imported?
- Was the proof script accepted by the proof assistant?
- What exactly was machine checked?
- What remains to be explained in ordinary mathematical language?
