# AI-Assisted Discovery Notes

AI-assisted mathematical discovery should be treated as candidate generation inside a verification workflow.

## Core distinction

AI can generate examples, conjectures, code, proof sketches, formal statements, and proof scripts. None of these should be treated as mathematical authority without an appropriate evidence standard.

## Working sequence

```text
Generate -> Test -> Prove -> Interpret
```

The machine may generate. The evaluator may test. The proof assistant may check. Humans remain responsible for meaning, novelty, significance, proof review, and responsible use.
