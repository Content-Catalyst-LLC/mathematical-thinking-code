# Haskell Companion Scaffold

This Haskell folder supports **Number, Pattern, and the Origins of Mathematical Thought**.

Haskell is useful here because units, tally marks, sequences, cycles, proof status, and pattern structures can be represented clearly with algebraic data types.

## What this folder demonstrates

- units and tally marks as typed objects;
- one-to-one correspondence;
- finite differences for sequence pattern detection;
- cyclic patterns with modular structure;
- proof-status distinctions for observed and proved patterns;
- structured CSV output for number-pattern workflows.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_sequence_audit.csv
../outputs/tables/haskell_tally_audit.csv
../outputs/tables/haskell_cycle_pattern.csv
```

## Professional note

These examples support mathematical cognition, sequence analysis, mathematics education, and responsible pattern interpretation. They do not replace proof, historical scholarship, formal verification, or expert mathematical review.
