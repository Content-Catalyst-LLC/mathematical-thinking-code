# Representation Notes

Mathematical representations preserve selected structure and omit other detail.

## Examples

- tally marks preserve unit count but omit object identity;
- tokens preserve unit equivalence but omit variation;
- numerals preserve quantity but omit collection context;
- diagrams preserve spatial relation but may omit proof and scale;
- formulas preserve general relation but may omit derivation;
- code preserves executable procedure but may omit human meaning.

Representation is not passive recording. It changes what can be remembered, shared, transformed, and reasoned about.
