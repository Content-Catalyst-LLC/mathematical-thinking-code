# Data Notes

The datasets are synthetic number-pattern and mathematical-origin metadata files.

- `raw/counting_objects.csv`: unit objects for tally and correspondence demonstrations
- `raw/sequences.csv`: arithmetic, quadratic, geometric, recursive, misleading, and cyclic sequences
- `raw/pattern_records.csv`: pattern examples, possible rules, proof status, and interpretation
- `raw/representation_practices.csv`: tally, token, numeral, diagram, formula, and code representations
- `raw/mathematical_origins.csv`: cognitive, cultural, and formal roots of mathematical thought
- `raw/cultural_practices.csv`: cultural practices and embedded mathematical forms
- `raw/interpretation_warnings.csv`: pattern-interpretation risks and mitigations
- `processed/`: generated data products
