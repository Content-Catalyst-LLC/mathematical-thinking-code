# Graph Algorithm Notes

Graph algorithms make graph structure executable.

## Common algorithms

- breadth-first search;
- depth-first search;
- Dijkstra-style shortest paths;
- topological sorting;
- connected components;
- minimum spanning tree;
- matching algorithms;
- centrality measures.

## Audit questions

- Does the algorithm match the graph type?
- Are edge weights nonnegative where required?
- Is the graph directed or undirected?
- Are cycles possible?
- Is a visited set required?
- What invariant proves correctness?
