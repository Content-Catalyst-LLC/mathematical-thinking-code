# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, degree audits, adjacency matrices, connected-component tables, shortest-path audits, directed-dependency audits, bipartite summaries, centrality interpretation indexes, algorithm indexes, and warning tables
- `outputs/figures/`: Graphviz DOT files and optional generated network diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
