# Data Notes

The datasets are synthetic mathematical metadata files used for reproducible demonstrations.

- `raw/sequence_patterns.csv`: finite sequence examples, expected pattern classes, and proof notes
- `raw/graph_edges.csv`: graph examples for invariant analysis
- `raw/pattern_records.csv`: pattern metadata with evidence type and proof status
- `raw/counterexamples.csv`: counterexample and false-pattern records
- `raw/dynamic_systems.csv`: dynamic-system examples for iteration
- `raw/probability_experiments.csv`: probability experiments for regularity under uncertainty
- `processed/`: generated data products
