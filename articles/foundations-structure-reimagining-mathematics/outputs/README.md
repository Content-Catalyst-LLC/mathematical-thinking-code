# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, foundation summaries, structure indexes, preservation-map matrices, formal-system summaries, model audits, proof-assistant layer audits, abstraction warnings, and adequacy checklists
- `outputs/figures/`: Graphviz DOT files and optional generated structure/transformation diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
