# Responsible Abstraction Notes

Abstraction selects what matters and omits what does not.

## Risks

- hidden assumptions;
- metric reduction;
- optimization harm;
- formal overconfidence;
- access barriers;
- category-level opacity.

## Principle

Formal correctness is not the same as ethical adequacy. Mathematics can prove, compute, optimize, or verify a statement while still leaving human responsibilities unresolved.
