# Data Notes

The datasets are synthetic mathematical representation metadata files.

- `raw/mathematical_objects.csv`: objects represented in multiple forms
- `raw/representations.csv`: representation records with preserved and omitted structure
- `raw/translations.csv`: translation relations among representations
- `raw/notation_warnings.csv`: representation risks and mitigations
- `raw/expressions.csv`: expression strings and representation types
- `raw/graph_edges.csv`: graph examples for edge-list and matrix translation
- `processed/`: generated data products
