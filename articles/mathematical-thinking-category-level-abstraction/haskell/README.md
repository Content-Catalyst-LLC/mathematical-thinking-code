# Haskell Companion Scaffold

This Haskell folder supports **Mathematical Thinking and Category-Level Abstraction**.

Haskell is useful here because category domains, morphism kinds, categorical concepts, abstraction risks, and review obligations can be modeled explicitly with algebraic data types.

## What this folder demonstrates

- categories as typed domains;
- morphism kinds as explicit structure-preserving arrows;
- categorical concepts as typed abstraction levels;
- abstraction risks as typed records;
- objects-arrows-structure-universality as a reusable review workflow;
- CSV output for reproducible audit artifacts.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_category_records.csv
../outputs/tables/haskell_abstraction_risks.csv
../outputs/tables/haskell_objects_arrows_structure_universality.csv
```
