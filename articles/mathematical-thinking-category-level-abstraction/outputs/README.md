# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, category indexes, functor summaries, natural transformation summaries, universal property summaries, commutative diagram checks, abstraction risk indexes, preservation/forgetting reviews, and responsible abstraction checklists
- `outputs/figures/`: Graphviz DOT files and optional generated category-abstraction workflow diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
