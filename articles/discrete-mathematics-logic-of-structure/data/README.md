# Data Notes

The datasets are synthetic finite examples for discrete mathematics and structure audits.

- `raw/discrete_objects.csv`: vertices, propositions, and finite states
- `raw/graph_edges.csv`: undirected, directed, and proof-dependency edges
- `raw/sequences.csv`: recurrence examples and proof-status notes
- `raw/combinatorics_cases.csv`: combinations, permutations, binary strings, and inclusion-exclusion
- `raw/boolean_cases.csv`: truth-table values
- `raw/algorithms.csv`: algorithm metadata, invariants, and complexity notes
- `raw/proof_patterns.csv`: direct proof, cases, contrapositive, induction, bijection, and invariants
- `raw/structure_warnings.csv`: responsible interpretation warnings for discrete structures
- `processed/`: generated data products
