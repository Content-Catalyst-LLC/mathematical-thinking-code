# Validation Plan

## Graphs

- Confirm vertices and edges are explicitly defined.
- Distinguish directed and undirected edges.
- Include isolated vertices.
- Separate graph structure from graph layout.

## Counting

- Identify whether order matters.
- Identify whether repetition is allowed.
- Check overlaps and double-counting.
- Use inclusion-exclusion where cases overlap.

## Recurrence and induction

- Define initial conditions.
- Verify recurrence generation.
- Label finite evidence separately from proof.
- Use induction for universal claims over natural numbers.

## Algorithms

- Document preconditions and invariants.
- Check termination and edge cases.
- Record complexity assumptions.
- Validate input structure.

## Responsible-use note

Synthetic examples in this repository are for methods demonstration. Applied use requires domain validation, stakeholder review, uncertainty documentation, and consequences analysis.
