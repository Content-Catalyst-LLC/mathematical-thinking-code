# Algorithm Notes

Algorithms are finite procedures that transform input into output.

## Audit dimensions

- input structure;
- output structure;
- preconditions;
- invariants;
- termination;
- correctness;
- complexity;
- edge cases;
- assumptions about data quality.

## Professional note

A correct algorithm can still produce harmful or misleading results if the encoded structure, input data, or decision context is flawed.
