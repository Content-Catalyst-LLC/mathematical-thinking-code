# Structure Ethics

Discrete structures can make systems legible, but they can also hide judgment.

## Risks

- categories can erase boundary cases;
- graph edges can imply unsupported relationships;
- Boolean rules can force rigid yes/no decisions;
- rankings can hide value judgments;
- algorithms can operationalize flawed assumptions;
- finite examples can be mistaken for proof.

## Responsible practice

Define units, relations, categories, rules, assumptions, edge cases, uncertainty, and affected communities before operationalizing a discrete structure.
