# Expression Tree Notes

Symbolic expressions can be represented as trees.

## Examples

- `x^2 + 2x + 1` is a tree with addition, power, multiplication, variables, and constants.
- `(x+1)^2` is a different tree representing an equivalent polynomial under valid assumptions.
- Factoring changes expression-tree form while preserving mathematical meaning under conditions.

Expression-tree representation is central to computer algebra systems.
