# Notation History Notes

Algebraic notation should be treated as intellectual infrastructure.

## Core historical layers

- procedural mathematics;
- rhetorical algebra;
- syncopated algebra;
- symbolic algebra;
- analytic geometry;
- structural algebra;
- symbolic logic and set notation;
- computer algebra and formal languages.

Modern notation can clarify older mathematics, but it can also distort it if reconstructed symbols are mistaken for original practice.
