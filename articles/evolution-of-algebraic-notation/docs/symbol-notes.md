# Symbol Notes

A symbol does not carry a single universal meaning outside context.

## Examples

- `x`: unknown, variable, coordinate, input, generic element.
- `=`: equality, equation, identity, definition, or assignment-like symbol in some contexts.
- `f(x)`: function application, not multiplication.
- `(G,*)`: abstract operation, not necessarily ordinary multiplication.
- `Ax=b`: matrix equation with hidden dimension and solvability assumptions.

Good notation pedagogy makes context explicit.
