# Transformation Notes

Algebraic transformation rules require assumptions.

## Examples

- Commutativity requires a commutative operation.
- Associativity requires an associative operation.
- Division by `a` requires `a != 0`.
- Exponent laws depend on domain and definition.
- Matrix inversion requires a square invertible matrix.
- Computer algebra simplification depends on domains, branch cuts, and assumptions.

Symbolic manipulation is not assumption-free.
