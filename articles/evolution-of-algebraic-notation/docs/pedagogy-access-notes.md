# Pedagogy and Access Notes

Compact notation can empower experts while excluding learners.

## Responsible practices

- Explain what each symbol means in context.
- Distinguish unknown, variable, coefficient, parameter, coordinate, and element.
- Use verbal, visual, tabular, and symbolic forms together.
- Make hidden domain assumptions explicit.
- Avoid treating modern notation as the only legitimate mathematical language.
- Include global histories of notation and algebraic expression.
