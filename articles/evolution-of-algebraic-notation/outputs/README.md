# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, notation-style summaries, milestone timelines, symbol-context audits, transformation-rule indexes, expression-tree audits, polynomial identity audits, warning indexes
- `outputs/figures/`: Graphviz DOT files and optional generated expression/notation diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
