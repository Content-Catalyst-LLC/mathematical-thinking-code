# Haskell Companion Scaffold

This Haskell folder supports **The Historical Understanding of Mathematics**.

Haskell is useful here because media, methods, historiographic risks, practices, and transmissions can be modeled explicitly with algebraic data types.

## What this folder demonstrates

- historical media as typed alternatives;
- mathematical methods as typed alternatives;
- historiographic risks as typed alternatives;
- practices as object-medium-method-meaning records;
- transmissions as preservation-plus-transformation records;
- CSV output for reproducible audit artifacts.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_historical_practices.csv
../outputs/tables/haskell_transmission_records.csv
../outputs/tables/haskell_historiographic_risks.csv
```

## Professional note

These examples support mathematical historiography, proof-style literacy, notation awareness, canon-risk review, and responsible historical interpretation. They do not replace expert history of mathematics, primary-source scholarship, philology, mathematical proof, or formal verification.
