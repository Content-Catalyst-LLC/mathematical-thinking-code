# Notation Notes

Notation is historical infrastructure.

## Examples

- tables organize repeated values and procedures;
- diagrams make spatial relations visible;
- place-value numerals scale arithmetic;
- algebraic notation makes unknowns manipulable;
- coordinates connect geometry and algebra;
- logical notation formalizes inference;
- code and proof scripts make procedures executable and derivations checkable.

Modern notation can clarify older mathematics, but it should not be mistaken for original practice.
