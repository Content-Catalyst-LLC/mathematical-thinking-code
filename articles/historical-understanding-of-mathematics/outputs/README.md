# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, object-medium-method-meaning indexes, proof-style catalogs, notation-history summaries, transmission summaries, historiographic risk indexes, canon-risk audits, and responsible interpretation checklists
- `outputs/figures/`: Graphviz DOT files and optional generated transmission diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
