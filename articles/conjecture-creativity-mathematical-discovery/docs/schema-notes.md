# Schema Notes

The SQL schema models mathematical discovery.

## Core tables

- `conjecture`: statements, domains, mathematical areas, proof status, and discovery notes.
- `evidence_record`: finite checks, examples, simulations, and counterexample evidence.
- `counterexample`: refutations and revision suggestions.
- `proof_attempt`: method, success/failure status, and lesson.
- `discovery_method`: method/risk/professional-practice metadata.

## Professional use cases

- theorem metadata;
- proof-status tracking;
- experimental mathematics documentation;
- counterexample catalogs;
- formalization planning;
- reproducible mathematical exploration;
- AI-assisted discovery audit trails.
