# Data Notes

The datasets are synthetic mathematical discovery metadata files.

- `raw/conjectures.csv`: conjecture statements, domains, proof status, and discovery notes
- `raw/evidence_records.csv`: examples, finite checks, simulations, and counterexample evidence
- `raw/counterexamples.csv`: refutations and revision suggestions
- `raw/proof_attempts.csv`: proof methods, status, and lessons
- `raw/sequence_terms.csv`: finite sequence examples for conjecture formation
- `raw/graph_edges.csv`: small graphs for invariant discovery
- `raw/discovery_methods.csv`: method/risk/practice metadata
- `processed/`: generated data products
