# Haskell Companion Scaffold

This Haskell folder supports **Conjecture, Creativity, and Mathematical Discovery**.

Haskell is especially useful here because conjectures, proof status, evidence records, counterexamples, and recursive mathematical structures can be represented naturally as algebraic data types.

## What this folder demonstrates

- algebraic data types for conjectures, evidence, and proof status;
- finite evidence audits;
- counterexample search;
- pure functions for conjecture testing;
- structured CSV output for reproducible discovery records.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_sum_finite_audit.csv
../outputs/tables/haskell_conjecture_records.csv
../outputs/tables/haskell_even_square_counterexamples.csv
```

## Professional note

These examples support formalization planning, proof-status discipline, mathematical discovery documentation, and reproducible exploration. They do not replace proof, proof-assistant checking, peer review, or expert mathematical interpretation.
