# Haskell Companion Scaffold

This Haskell folder supports **Geometry and the Visual Mind in Mathematics**.

Haskell is useful here because geometric objects, transformations, invariants, and visual-reasoning audit records can be represented clearly with algebraic data types.

## What this folder demonstrates

- points and shapes as typed data;
- transformations as explicit constructors;
- distance, area, orientation, and unit-circle residual checks;
- transformation audits for preserved and changed structure;
- structured CSV output for geometric reasoning workflows.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_point_membership.csv
../outputs/tables/haskell_triangle_measurements.csv
../outputs/tables/haskell_transformation_audit.csv
```

## Professional note

These examples support geometric reasoning, computational geometry, visualization audit trails, and mathematics-learning research. They do not replace proof, formal verification, mathematical expertise, or contextual interpretation.
