# Data Notes

The datasets are synthetic geometric and visual-reasoning metadata files.

- `raw/geometric_objects.csv`: points, segments, triangles, polygons, circles, graphs, and topological objects
- `raw/points.csv`: coordinate data used in distance, area, and residual checks
- `raw/shapes.csv`: shape-level metadata and expected properties
- `raw/transformations.csv`: transformation types, preserved structures, and risks
- `raw/representations.csv`: diagram, coordinate, equation, and graph representations
- `raw/diagram_warnings.csv`: visual-representation risks and mitigations
- `raw/invariants.csv`: structures preserved by transformation classes
- `raw/visualization_risks.csv`: visual reasoning risks and checks
- `processed/`: generated data products
