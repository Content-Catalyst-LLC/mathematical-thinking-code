# Visualization Ethics

Visual representations can clarify complexity, but they can also distort.

## Common risks

- map projection distortion;
- graph layout misleading distance;
- simulation realism exceeding model reliability;
- high-dimensional projection loss;
- AI-generated false structure;
- special-case diagrams presented as general truth.

## Responsible practice

State assumptions, scale, data sources, transformation rules, projection methods, uncertainty, and what the representation preserves or omits.
