# Schema Notes

The SQL schema models geometric and visual-reasoning metadata.

## Core tables

- `geometric_object`: points, segments, triangles, polygons, circles, graphs, and topological objects.
- `geometric_representation`: diagrams, coordinates, measurements, equations, graph drawings, and topology descriptions.
- `transformation_audit`: transformation classes, changes, preserved structures, and risks.
- `diagram_warning`: visual reasoning warnings and mitigations.
- `invariant`: structures preserved under transformations.
- `visualization_risk`: broader visual interpretation risks and responsible checks.

## Professional use cases

- geometry education;
- visual proof analysis;
- computational geometry;
- coordinate representation audits;
- diagram and visualization review;
- transformation and invariance analysis;
- AI-generated visual explanation audits.
