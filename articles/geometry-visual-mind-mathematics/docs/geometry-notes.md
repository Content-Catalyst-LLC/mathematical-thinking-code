# Geometry Notes

This article treats geometry as reasoning about shape, space, relation, transformation, and invariance.

## Core distinctions

- diagrams can guide proof but do not automatically prove;
- coordinate representations make geometry computable but can hide visual intuition;
- transformations reveal what changes and what is preserved;
- topology abstracts from metric shape toward connectivity and holes;
- computational visualization requires interpretation and validation.

## Professional use

The repository can support geometry education, computational geometry, visual proof analysis, transformation audits, diagram-risk documentation, and formalization planning.
