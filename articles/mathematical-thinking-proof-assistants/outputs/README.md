# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, proof-assistant system summaries, formalization project indexes, theorem-statement audits, proof-layer summaries, trust-boundary maps, AI workflow audits, skill indexes, and responsible verification checklists
- `outputs/figures/`: Graphviz DOT files and optional generated proof-assistant workflow diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
