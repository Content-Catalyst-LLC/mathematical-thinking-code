# Responsible Verification Notes

Machine-checked proof is powerful, but responsible use requires interpretation.

## Questions

- What exactly was proved?
- Which assumptions and axioms were used?
- Which libraries were imported?
- Does the formal statement match the intended claim?
- What should not be inferred from the proof?
- How might this theorem or verified system be used in the world?
- Who can reproduce, understand, or challenge the result?

Machine-checked proof is not the same as responsible use.
