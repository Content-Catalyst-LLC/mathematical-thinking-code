# AI-to-Proof-Assistant Workflow Notes

AI can support proof-assistant work, but it should be treated as proposal generation.

## Responsible workflow

```text
AI proposes -> proof assistant checks -> human interprets
```

## Review questions

- Did AI generate a valid statement?
- Do the hypotheses match the intended theorem?
- Does the proof assistant accept the derivation?
- What theorem was actually proved?
- Does the formal proof explain the mathematical idea?
- What remains a human judgment?

The authority comes from checking and interpretation, not from AI fluency.
