# Data Notes

The datasets are synthetic finite examples for set, relation, and function audits.

- `raw/sets.csv`: finite domains, codomains, classification labels, and definition rules
- `raw/relation_pairs.csv`: function graphs, equivalence relations, partial orders, and non-function relations
- `raw/functions.csv`: mapping definitions, domain/codomain notes, and validation examples
- `raw/equivalence_examples.csv`: equivalence-class examples and proof-status labels
- `raw/mapping_warnings.csv`: boundary, codomain, missing-input, category, and multivalued warnings
- `raw/modeling_examples.csv`: toy modeling examples that combine sets, relations, functions, and assumptions
- `processed/`: generated data products
