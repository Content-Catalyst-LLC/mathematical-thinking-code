# Relation Notes

Relations are modes of mathematical thought about connection.

## Relation properties

- reflexive: every element relates to itself;
- symmetric: relation works in both directions;
- transitive: relation chains through intermediates;
- antisymmetric: mutual relation implies equality;
- total or partial: not all elements must be comparable.

## Common structures

- equivalence relations;
- partial orders;
- graph adjacency;
- divisibility;
- similarity;
- dependency;
- accessibility.

Relations should be defined before they are interpreted.
