# Haskell Companion Scaffold

This Haskell folder supports **Mathematical Thinking and the Ethics of Quantification**.

Haskell is useful here because metric types, consequence levels, metric risks, and review obligations can be modeled explicitly with algebraic data types.

## What this folder demonstrates

- metric records as typed structures;
- consequence level as an explicit governance requirement;
- metric risks as typed categories;
- define-measure-contextualize-govern as a reusable workflow;
- CSV output for reproducible audit artifacts.

## Run

```bash
cd haskell
make run
```

This writes generated artifacts to:

```text
../outputs/tables/haskell_metric_records.csv
../outputs/tables/haskell_define_measure_contextualize_govern.csv
```

## Professional note

These examples support metric ethics audits, proxy validity review, Goodhart risk checks, ranking and aggregation review, uncertainty reporting, justice review, and responsible metric governance. They do not replace domain expertise, empirical validation, community review, legal review, ethical review, qualitative evidence, institutional accountability, or public deliberation.
