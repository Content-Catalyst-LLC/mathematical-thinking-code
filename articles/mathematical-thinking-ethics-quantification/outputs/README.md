# Outputs

Generated outputs should be written here.

- `outputs/tables/`: CSV files, SQLite databases, metric indexes, risk summaries, validity reviews, governance checks, Goodhart summaries, aggregation reviews, ranking reviews, uncertainty reviews, justice reviews, and responsible quantification checklists
- `outputs/figures/`: Graphviz DOT files and optional generated quantification ethics workflow diagrams

These outputs are reproducible artifacts created by the scripts in the language-specific folders.
