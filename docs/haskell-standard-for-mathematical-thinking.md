# Haskell Standard for Mathematical Thinking Repositories

Haskell should be treated as a first-class language in Mathematical Thinking article scaffolds when the article involves logic, proof, abstraction, recursion, formal inference, type theory, category-style structure, proof assistants, theorem metadata, or computation-as-reasoning.

## Standard folder

```text
haskell/ — algebraic data types, formal inference, proof trees, recursion, type-structured examples, and functional mathematical modeling
```

## Recommended uses

- propositions and logical connectives as algebraic data types;
- truth tables and inference-rule evaluation;
- proof trees and derivation graphs;
- recursive sequence and structural induction examples;
- graph invariants and dependency analysis;
- type-class examples for algebraic abstraction;
- Curry–Howard-style demonstrations where appropriate;
- formalization planning for proof assistants.

## Repository practice

Future scaffolds should include Haskell more prominently, especially in articles about logic, proof, formal inference, abstraction, category-like structure, recursion, proof assistants, and mathematical computation.
